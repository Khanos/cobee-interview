## About The Template

It's a basic Model View Controller template as base for backend server using Nodejs (dhaa).

## The Tech

* Nodejs
* Express
* Typescript
* Mocha/chai/supertest

## How to use

> git clone git@github.com:Khanos/nodejs-backend-template.git

> npm install

> npm run serve
